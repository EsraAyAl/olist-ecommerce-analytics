#  E-commerce Data Analytics Project

##  Project Overview
This project focuses on analysing an e-commerce dataset to understand revenue performance, delivery efficiency, and customer satisfaction.

The analysis was carried out using **Python, SQL, and Power BI**, combining data exploration, transformation, and interactive dashboarding.

---

##  Objectives
- Analyse overall revenue and order trends
- Identify top-performing product categories
- Evaluate delivery performance and late deliveries
- Understand the impact of delivery delays on customer satisfaction

---

##  Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI
- Excel (data source)
